Обо мне

Примеры кейсов

Интеллектуальный ассистент для обработки email

Создан Apps-скрипт, который использует API языковой модели (ИИ) для анализа и классификации входящих писем. Робот автоматически определяет тему запроса и вносит данные в гугл таблицу для последующей обработки менеджером.

Технологии: Apps Script, AI API (GPT/Gemini), REST API

Автоматизация отчетности в Google Sheets

Разработан скрипт в среде Google Apps Script, который ежедневно собирает данные из различных источников и формирует сводный отчет. Это экономит до 5 часов ручной работы еженедельно.

Технологии: Google Apps Script, REST API, Google Sheets

Парсер сайтов школ и автоматическая рассылка

Разработана RPA-система для сбора данных. Python-скрипт парсит информацию с сайтов школ (контакты, ФИО директора), а Google Apps Script используется для автоматической отправки первичных писем для установления контакта.

Технологии: Python, Google Apps Script, RPA
